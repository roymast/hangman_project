# HangmanPython
 Hello there!
 this is my multi-player Hangman game! 
# Requirements
 - Python 3.7.x
 - Pygame 2.0.x
 - EasyGui 0.98.3
# How to use
 1. Clone the repo
 2. Install the required libraries.
 3. Run Server.py and than Client.py(reccomend using multiple computers, must be on the same network)
